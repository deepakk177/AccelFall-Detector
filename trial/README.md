## 🛠️ Customization

To customize this portfolio for your own use:

1. **Update Content**: Modify the text, project descriptions, and contact information in `index.html`
2. **Change Colors**: Edit CSS variables in the `:root` selector in `styles.css`
3. **Adjust Particles**: Modify `particleCount` in `script.js` for more/fewer particles
4. **Add Projects**: Add more project cards in the projects section
5. **Update Skills**: Modify skill categories and progress percentages
